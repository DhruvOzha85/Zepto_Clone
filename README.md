# 🛒 Zepto Clone

A static front-end clone of the **Zepto** quick-commerce grocery delivery app, built with pure HTML and CSS. This project recreates the core pages of Zepto's website including the home page, product listings, categories, and about section.

🔗 **Live Demo:** [clone-zepto.netlify.app](https://clone-zepto.netlify.app/)

---

## 📁 Project Structure

```
Zepto_Clone/
├── index.html          # Home / Landing page
├── products.html       # Products listing page
├── categories.html     # Product categories page
├── about.html          # About Zepto page
└── styles.css          # Global stylesheet
```

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Page structure & content (82.0%) |
| CSS3 | Styling & layout (18.0%) |

---

## ✨ Features

- 4-page static website inspired by Zepto's UI
- Product listing and category browsing pages
- Clean, modern e-commerce style layout
- Fully static — no frameworks or JavaScript required
- Deployed and publicly accessible via Netlify

---

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/DhruvOzha85/Zepto_Clone.git
   ```
2. Open `index.html` in your browser — no build step or dependencies required.
